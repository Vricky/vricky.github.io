树莓派使用基本设置

2016-12-05

首先，树莓派正常使用必须品：

树莓派板子，通电电源、线，内存卡（卡内须有系统）。这几项足够让你的树莓派开启。关于鼠标，键盘，显示器之类的都是辅助用品。卡内系统的安装问题，等待自己研究一下后再进行总结。可先用自带系统烤入内存卡进行使用。

树莓派默认系统文字为英文，且中文显示为乱码，所以开启后还需配置中文字库，方法如下：

1.安装中文字体。
终端下执行命令：sudo apt-get -y install ttf-wqy-zenhei

2.安装中文输入法
终端下执行命令：sudo apt-get -y install scim-pinyin

3.设置显示中文。
安装完成后，终端下执行命令：sudo raspi-config 调出系统配置界面
接着操作提示：
选择 5 Internationalisation Options
选择I 1 Change Locale
选择 zh-CN，UTF8

4.重启生效。终端下执行命令：sudo reboot 重启。

可以给树莓派设置远程连接和远程桌面设置：

终端下执行命令： ifconfig-a，查看树莓派的IP。

两种远程连接方式：
A：SSH远程连接，建议使用putty连接链接输入树莓派IP，用户名：pi 密码raspberry，就出现终端界面了。
B：远程桌面连接
终端下执行命令：sudo apt-get install xrdp ，安装远程桌面支持。
在windows电脑上点开始，输入mstsc，出来远程桌面界面，输入树莓派IP，用户名：pi 密码raspberry，就可以连上了。
终端下执行命令：sudo poweroff 关机。然后断开电源。

这样树莓派对你来说就只需要上述必需品即可在自己的个人电脑上利用远程连接来进行图形化操作。

关于树莓派音频输出问题：

树莓派其默认音频输出为自动。树莓派有两种品拼输出模式：HDMI（2）和3.5耳机接口（1）输出。可以根据需要进行间切换。

以下为三种切换方式：

1.在命令行终端上输入下面命令将会切换到HDMI输出模式：

amixer cset numid=3 2

这里将输出设置为2，也就是HDMI。

将输出设置为1将切换到模拟信号（也就是耳机接口）。
默认的设置为0，代表自动选择。

2.在raspi-config设置工具下

2.1命令行打开raspi-config工具
2.2选择advanced Options进入，并选音频audio选项

2.3现在你可以看到默认选择Auto选项以及另外两种模式的列表。选择其中一种模式然后回车，再按向右光标控制键跳出选项菜单，选择Finish退出配置工具，完成配置。

3.在一些少见的情况下，你需要编辑config.txt文件来强制进入HDMI模式（为了排除没有附带声音的DVI模式）。实现这个目的，你可以编辑/boot/config.txt文件设置hdmi_drive=2，然后重启系统让修改生效。

重点提醒：如果你浏览器下尝试进行音频输出（就是传说中的听歌啦），可能会有很多杂音，这个要淡定，楼主也因为这件事搞了一上午才搞清楚，你使用的浏览器一定是firefox或者是年轻的使用了自带的浏览器，可能是因为这两个浏览器的音频解码问题，对mp3格式的解码貌似有点问题，这个时候强烈建议使用chromium浏览器，就是你windows下常用的chrome。树莓派已经支持HTML5音视频的Chromium浏览器安装。chromium浏览器和谷歌浏览器使用一样的webkit核心，速度简直跟firefox不是一个等级，渲染效果也是挺好的。安装方法如下：

cd ~
wget https://launchpadlibrarian.net/2 ... 04.1.1183_armhf.deb
wget https://launchpadlibrarian.net/2 ... 04.1.1183_armhf.deb
wget https://launchpadlibrarian.net/2 ... 04.1.1183_armhf.deb
sudo apt-get insall libgnome-keyring0 xdg-utils libss3
sudo dpkg -i *.deb

注：第五步可能存在libss3的安装有点问题，可以忽略继续进行下一步。安装完成后就可以在Internet下找到chromium进行使用，在线听歌啥的都是洒洒水。
